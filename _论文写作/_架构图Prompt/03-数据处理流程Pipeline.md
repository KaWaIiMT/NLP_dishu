# 图C：数据处理与分析流程Pipeline图

## 图类型
流程图 / Pipeline Diagram（数据从左到右流动）

## 风格要求
- 简洁的横向流程图
- 配色：蓝灰底色调
- 三个阶段用不同背景色块区分
- 箭头连接
- 字体：无衬线

## 图表内容

### 标题
"数据处理与分析流程" / "Data Processing and Analysis Pipeline"

### 三大阶段（横向排列，从左到右）

#### 阶段一：数据准备（蓝色背景 #D6EAF8）
包含三个步骤（从上到下或左右排列）：
1. "原始标注数据\n708文件"
2. "数据清洗与验证\n剔除不合规文件"
3. "标注提取与分组\n按task_id提取标注值"

→ 箭头指向阶段二

#### 阶段二：特征提取（绿色背景 #D5F5E3）
包含两个并行分支：
- 上方分支："地书侧\n• literal_gloss → token序列\n• POS/篇章/形态 → 类别分布"
- 下方分支："自然语言侧\n• jieba分词 → token序列\n• 降采样对齐(token数统一)"

→ 箭头指向阶段三

#### 阶段三：对比分析（橙色背景 #FDEBD0）
包含三个分析模块：
1. "词汇层面\nZipf/熵/TTR"
2. "语法层面\nPOS分布/形态特征"
3. "篇章层面\n篇章关系分布"
4. "标注者间一致性\nCohen's κ / Jaccard"

→ 最终输出

#### 输出（紫色背景 #E8DAEF）
"六维度语言性评估报告\n+ 歧义热力图"
