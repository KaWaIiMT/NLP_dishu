# Task2：地书-自然语言复合词元生成

## 概述
根据地书JSON标注数据，结合字面翻译（Literal Gloss）和语义角色标注，生成地书-自然语言复合词元。

## 方法
1. 读取地书JSON标注文件
2. 提取词单元标注、字面翻译和语义角色
3. 生成复合词元：格式为 [词性]_[字面翻译]_[语义角色]
4. 保存结果为JSON和文本格式

## 文件结构
```
Task2-复合词元生成/
├── 说明文档.md          # 本文件
├── 复合词元生成器.py    # 主要处理脚本
└── 生成结果/
    ├── 词标注复合词元.json
    ├── 词标注复合词元.txt
    ├── 句标注复合词元.json
    └── 统计报告.md
```

## 运行方式
```
python 复合词元生成器.py
```
