# 任务组A：数据准备

## 概述
本文件夹包含研究计划中任务组A的相关工作，包括数据预处理和复合词元生成。

## 任务列表

### Task 1：数据清洗与准备
- **状态**：已完成
- **说明**：对地书标注数据进行了清洗，分离了词标注和句标注
- **结果**：保存在 `原始语料链接/` 文件夹中（通过相对路径指向原始数据）

### Task 2：复合词元生成
- **状态**：进行中
- **说明**：根据地书JSON标注数据，结合Literal Gloss和语义角色标注，生成地书-自然语言复合词元
- **位置**：`Task2-复合词元生成/`

### Task 3：语料库采样与划分
- **状态**：待处理
- **说明**：从多种自然语言数据中采样，构建训练/验证/测试集，以及不同语体的划分
- **位置**：`Task3-语料库采样/`

## 数据来源

### 地书数据
- `地书实验结果/清洗后数据/词标注/` - 地书词标注数据
- `地书实验结果/清洗后数据/句标注/` - 地书句标注数据

### 自然语言对照数据
- `素材/THUcNews/` - 中文新闻语料
