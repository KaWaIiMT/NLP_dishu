# 分组任务（11~16周）

#### 要求

- 分组完成任务，每组不超过4人
- 每组可从下面指定的多个“分组作业主题”中选择一个完成
- 每组评分完全按照整组的成果来衡量
    - 意味着：若人数不足4人，从“工作量”角度而言有自然劣势，但由于免去了协调难度，有可能特定目标下，人数更少效率更高，因此大家需要根据本组想要做的任务的性质来进行分组。
- 各组可自行扩展“地书标注工具 V1.0”，以获得更有独特价值的标注数据，还可借助任意大模型来获取标注信息。
- 各组在各个共工作环节中，应充分运用大语言模型，特别是不应局限于“提问/聊天”

#### 分组作业主题1-“地书vs自然语言”主题论文

周次：第11周到第16周（实验课）

- 任务概述
    
    完成一篇学术论文，主题为“地书”的图形语言与实际语言文本的异同，应选取特定视角（例如从词义、词性、句法结构等语言学中的视角），通过对“地书”原始图形、“地书”的标注、其他自然语言文本素材的分析来研究，分析方法应该包括Transformer以前的经典NLP方法，以及基于Transformer的大语言模型方法。
    
- 评价标准
    
    研究问题独特性及意义 x 综合品质
    
- 演示
    
    有意愿公开演示成果的组，将安排在第12~16周实验课上进行展示，时间允许可优化并再次演示。
    
    演示本身并不算分，但有助于优化作业，并且高质量演示会有少量作业加分。
    
- 提交
    
    提交内容：
    
    文稿（docx）、数据分析结果汇集(xlsx)、演示材料（PPT、PDF、图片、视频）
    
    提交链接:
    
    论文：
    
    【腾讯文档】NLP期末作业——论文
    https://docs.qq.com/form/page/DTU1rQ1JuQXFOQ1VD
