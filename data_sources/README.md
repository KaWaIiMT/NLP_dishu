# EarthScript Paper Data Sources

The user has provided the following datasets for analysis:

**EarthScript (地书) Data:**
1. `地书标注系统_V1.0.zip` (Manual annotation tool)
2. `《地书》标注数据.zip` (Basic words/sentences annotation data)
3. `《地书》白话版标注数据.zip` (Colloquial translation annotation data)

**Natural Language Corpora for Comparison:**
1. DaZhiGe Ancient Chinese (`四库全书数据库.zip` - Siku Quanshu)
2. World Literature in Modern Chinese (`世界名著.zip`)
3. Chinese News Corpus (`THUcNews.zip`)
4. Modern Chinese Novels (`小说.zip`)
5. Multilingual Wiki Sentences (`WikiSentences30k_16Lans.zip` - 30k sentences each across 16 languages)
